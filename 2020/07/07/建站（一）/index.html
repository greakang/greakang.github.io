使用hexo+github搭建免费个人博客

前言

  • 什么是hexo：

    Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页

  • 使用github pages服务器搭建博客的优点：

    1. 全是静态文件，访问速度快；
    2. 免费方便，不用花一分钱就可以搭建一个自由的个人博客，不需要服务器不需要后台；
    3. 可以随意绑定自己的域名，不仔细看的话根本看不出来你的网站是基于github的；
    4. 数据绝对安全，基于github的版本管理，想恢复到哪个历史版本都行；
    5. 博客内容可以轻松打包、转移、发布到其它平台；

安装Node.js

首先安装node.js可以根据自己的操作系统选择合适的版本。

安装选项全部默认，一直点击NEXT就可以了。

安装好之后，按win+r输入cmd打开命令提示符，输入node -v和npm -v。如果出现版本号，就安装成功了。

添加国内镜像源

如果没有稳定的梯子的话，可以使用阿里云的国内镜像进行加速,在命令提示符中输入下列代码：
npm config set registry https://registry.npm.taobao.org

安装git

为了把本地的网页文件上传到github上面去，我们需要用到分布式版本控制工具—Git

安装选项还是全部默认

安装完成后在命令提示符中输入git –version验证是否成功。

注册github

在github中新建一个项目：

然后如下图所示，输入自己的项目名称，注意名称要和你的github名字一样，比如你github的名字是abc那么仓库的名字也一定是abc.github.io还有README初始化文件也一定要点上。

然后项目就算建好了，点击项目的setting，下拉到最后有个Github Pages，点击 choose a theme选择一个主题。然后回到Github Pages 点击下面的链接就可以进入自己的网页了。效果类似下图：

安装hexo

在自己电脑的一个盘里建立一个文件夹以后就是放自己网页的配置文件了。建好文件夹以后进入文件夹，点击右键 git Bash here 打开git的控制台。

然后输入npm i hexo-cli -g安装hexo

安装好之后输入hexo -v看是否安装成功。

接下来继续输入hexo init初始化文件夹，接着输入npm install安装必要的插件

最后输入hexo g生成静态网页，接下来输入hexo s打开自己的浏览器输入http://localhost:4000/,就可以在本地看到我们的博客啦

连接github与本地

之前我们生成的网页其实是在本地生成的，现在我们需要讲本地生成好的网页上传到github让其他人也可以看到。

随便一个位置右键打开git bash here，然后输入以下命令：
git config --global user.name "用户名"

git config --global user.email "邮箱"

用户名和邮箱根据你注册github的信息自行修改。

然后生成密钥SSH key：

ssh-keygen -t rsa -C "邮箱"

打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。

git bash中输入

cat ~/.ssh/id_rsa.pub

将输出的内容复制到框中，点击确定保存。

输入ssh -T git@github.com，如果如下图所示，出现你的用户名，那就成功了。
